Display: Flex

Flexbox is short for the flexible box Layout module. Flexbox is a layout method for arranging items in rows or colums. Flexbox makes it easily to design a flexible respomsive layout structure, without using float or positioning.

    • nowrap
    • wrap
    • wrap-reverse

Display: Flex, Flex - Wrap: no-wrap

Display: Flex, flex-Direction

  • The flex-direction property specifies the display-direction of flex items in the flex container.
  • The flex-direction property can have one of the following values.
    • row
    • column
    • row-reverse
    • column-reverse

Display: Flex, flex-Direction : row

  • The row value is the default value, and it displays the flex items horizontally

Display: Flex, flex-flow

  • The flex-flow property is a shorthand property foe setting both the flex-Direction and flex-Wrap property
  • The flex-flow property can have one of the following values
    • row-wrap
    • row-wrap-reverse
    • row-nowrap
    • column-wrap
    • column-wrap-reverse
    • column-nowrap

Display: Flex, flex-flowrow wrap;

  • The CSS property flex-flow row wrap is a shorthand for setting both the flex-direction and flex-wrap property on a flex container
